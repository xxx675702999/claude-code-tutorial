## 🚀 快速开始

### 什么是 Claude Code？

Claude Code 是 Anthropic 开发的 AI 辅助编程工具，通过插件系统可以扩展其功能。本教程将教你如何从零开始开发自己的插件。

### 适用人群

- 🎯 想要扩展 Claude Code 功能的开发者
- 🔧 对插件开发感兴趣的技术爱好者
- 📚 希望深入理解 Claude Code 架构的工程师
- 🌍 需要中英双语技术文档的学习者

### 你将学到

- ✅ Claude Code 的核心架构和设计理念
- ✅ 如何开发自定义工具
- ✅ Hooks 系统的使用方法
- ✅ 插件系统的完整流程
- ✅ 错误处理和性能优化
- ✅ 生产级插件的开发实践
