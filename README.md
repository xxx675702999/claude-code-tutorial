## 📚 项目简介

这是一套完整的 Claude Code 源码教学指南，包含 23 章详细教程，376 个代码示例，中英双语版本。

### ✨ 特色

- 📖 **内容完整** - 23 章从入门到实战的完整教程
- 💻 **代码丰富** - 376 个验证过的代码示例
- 🌍 **双语支持** - 中英文双语，术语对等 100%
- 🚀 **实战导向** - 包含完整的实战项目
- 🔍 **深入源码** - 深入分析核心原理
- ⚡ **在线阅读** - 基于 VitePress 构建的现代化文档站点

## 🚀 快速开始

### 在线阅读

访问 [GitHub Pages](https://your-username.github.io/claude-code-tutorial/) 阅读完整教程。

### 本地运行

```bash
# 克隆仓库
git clone https://github.com/your-username/claude-code-tutorial.git
cd claude-code-tutorial

# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 访问 http://localhost:5173
```
